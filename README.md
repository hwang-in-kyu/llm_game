# 🏢 퇴근 설득 게임 (LLM 기반 인터랙티브 게임)

회사 상사를 설득해 퇴근을 성공시키는 텍스트 기반 AI 게임입니다.  
LLM의 프롬프트 엔지니어링을 활용하였습니다.

예시:
- 상황: 상사가 오늘 유독 예민함
- 상사: FM군인형
- 결과: 설득력 50% ~

## ✨ 주요 기능

- 🎭 다양한 상사 캐릭터 (꼰대형, 논리형, 감정폭발형 등)
- 📌 랜덤 상황 생성 (회식, 마감, 회의 등)
- 🧠 LLM 기반 실시간 대화
- 📊 설득력 점수 시스템 (0~100)
- ⚖️ 일관성 기반 판정 로직
- 🚨 모순 발생 시 즉시 실패
- 🎯 턴 기반 게임 구조

## 🎯 게임 규칙

- 총 5턴 동안 상사를 설득해야 합니다
- 첫 턴은 반드시 "안녕하십니까"로 시작해야 합니다
- 매 턴 설득력이 변동됩니다
- 마지막 턴에서 성공/실패 판정

### ✔ 성공 조건
- 설득력 80 이상

### ❌ 실패 조건
- 설득력 30 이하
- 모순 2회 이상
- 말 바꿈 발생

## 🛠 기술 스택

- Python
- Streamlit
- OpenAI API (GPT-4.1-mini)
- Prompt Engineering
